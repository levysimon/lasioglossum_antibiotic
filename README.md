# Antibiotic impact on gut microbiota and social behavior in the wild bee *Lasioglassum Malachurum*

We used a combination of behavioral analysis, activity analysis and RNA-sequencing to investigate the effect of antibiotic on the gut microbiota and social behavior in the wild bee Lasioglossum malachurum

**Statistical Analysis Pipeline**

This repository contains the complete R analysis pipeline for investigating how antibiotic treatments and subsequent gut microbiota re-inoculation affect the activity and behavior of *Lasioglassum Malachurum* wild bees.
