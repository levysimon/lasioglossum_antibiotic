## Activity and Behavior Analysis

* **Raw Video Tracking Logs:** Must be stored under `/data/behavior_csv_raw/` with files named as `[Year]_petridish[Number].csv`.
* **Required Data Frames:** `moving.csv` and `inderindividual_dist.csv` are located in the `/data/animal_ta/` directory.
* **Outputs:** All generated tables, model prediction plots, and multivariate statistics files are saved inside the `/result/` subdirectory.
