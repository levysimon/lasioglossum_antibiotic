# Amplicon sequencing 

The SILVA reference databases should be downloaded from: https://zenodo.org/record/3986799#.YcW1qy1h2gQ

You can either use the ps_dada2taxa_Lasio.rds file (inside the /data directory) or download the raw sequencing data (deposited at the SRA Database under accession no. SRP666708). 

All other required datasets and intermediate files can be found in the /data directory.
